# Itda(잇다)
성공회대 소프트웨어공학과 2018 캡스톤디자인 수업 프로젝트

##프로젝트 개요
-개발 시스템 설계 내용 : 교내 오프라인 모임을 위한 온라인 공유 및 실시간 타겟팅 알림 플랫폼
-시스템 이름 : Itda(잇다)
-역할 : 교내 오프라인 모임 중 정보 공유의 편리성 증진 및 실시간 타겟팅 알림으로 정보 전달률 향상
